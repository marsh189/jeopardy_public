HOW TO PLAY JEOPARDY

- Enter a namer to be known as
- Once all 3 contestants join, The server will send the first 'Answer'
- To buzz in, click 'Enter' to buzz in
- The first person to 'buzz in' will be allowed to enter a 'Question'
- If the input is right, the server will send out a new 'Answer'
- If the input is wrong, the server will notify the contestants that the other two can buzz in to answer
